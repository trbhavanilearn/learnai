# 📦 Installing Required Libraries

## 1. Install OpenCV

OpenCV is one of the most popular Computer Vision libraries.

Install using:

```bash
pip install opencv-python
```

Import:

```python
import cv2
```

Purpose:

* Read Images
* Capture Webcam
* Image Processing
* Video Processing
* Object Detection
* Face Detection

---

## 2. Install EasyOCR

EasyOCR is an Optical Character Recognition (OCR) library.

Installation:

```bash
pip install easyocr
```

Import:

```python
import easyocr
```

Purpose:

* Read text from images
* Extract printed text
* Read scanned documents
* Read number plates
* Read receipts

Example Applications:

* Invoice Reader
* Passport Scanner
* Document Digitization
* Number Plate Recognition

---

## 3. Install Ultralytics

Ultralytics provides the implementation of the YOLO (You Only Look Once) object detection models.

Installation:

```bash
pip install ultralytics
```

Import:

```python
from ultralytics import YOLO
```

Purpose:

* Object Detection
* Image Classification
* Image Segmentation
* Pose Detection
* Object Tracking

Example:

```python
from ultralytics import YOLO

model = YOLO("yolov8n.pt")
```

Example detections:

* Person
* Car
* Bus
* Truck
* Dog
* Cat
* Bicycle

---

## 4. Install Faster Whisper

Faster Whisper is a Speech-to-Text library based on OpenAI's Whisper model.

Installation:

```bash
pip install faster-whisper

Import:

```

Purpose:

* Convert speech into text
* Audio transcription
* Meeting transcription
* Subtitle generation
* Voice assistants

Example Workflow:

```text
Audio

Speech Recognition
   ↓
Text Output
```

Applications:

* YouTube subtitles
* Meeting notes
* Interview transcription
* Voice search

## 5. Install Edge TTS

Text
  ↓

* Multiple Languages

# Installing Linux Dependencies for OpenCV

Sometimes OpenCV may display the following error:

```text
install libgtk2.0-dev and pkg-config, then re-run cmake or configure script in function 'cvShowImage'
```

This error occurs because OpenCV requires GUI libraries to display images using functions such as:

```python
cv2.imshow()
```

---

## Ubuntu / WSL Installation

Install the required packages:

```bash
sudo apt update
```

```bash
sudo apt install libgtk2.0-dev
```

```bash
sudo apt install pkg-config
```

Verify installation:

```bash
pkg-config --version
```

After installing the dependencies, restart your terminal and rerun your Python application.

# What is an LLM?

LLM stands for:

```text
Large Language Model
```

A Large Language Model is an Artificial Intelligence model trained on massive amounts of text data to understand and generate human-like language.

---
